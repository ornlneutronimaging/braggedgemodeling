Texture¶

class bem.xtaloriprobmodel.MarchDollase(r=None, beta=None)¶

March Dollase model for texture

Create an instance

>>> md = MarchDollase()

Now md.r and md.beta are parameters. The default value for r is 1. The default value for beta is 0.

To change r and beta value for a specific hkl, do

>>> md.r[hkl] = new_r_value
>>> md.beta[hkl] = new_beta_value
